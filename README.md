# MkulimaKe

A single-page site and app for keeping farm livestock and poultry records — animal profiles, health/vaccination logs, weight tracking, and a dashboard, all in one self-contained HTML file.

## Deploy

This is a static site with no build step. To deploy on Vercel:

1. Push this repo to GitHub.
2. In Vercel: **Add New → Project → Import** this repo.
3. Framework preset: **Other** (no build command needed).
4. Deploy.

`index.html` is the entry point.
